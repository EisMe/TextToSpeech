# ქართული ტექსტის მეტყველებაში გადამყვანი (TTS) სისტემა

**წესებზე დაფუძნებული ქართული ტექსტიდან მეტყველებაში გადამყვანი (TTS) სისტემა**, რომელიც აგენერირებს აუდიოს ქართული ტექსტიდან **მარცვლების კონკატენაციის** მეთოდით. ეს პროექტი მიზნად ისახავს ქართული ენის უნიკალური ფონეტიკური და მორფოლოგიური მახასიათებლების დამუშავებას.

---

## მახასიათებლები

- მარცვლებზე დაფუძნებული კონკატენაციური მიდგომა.
- მხარდაჭერილი შეყვანის მეთოდები:
  - ტექსტის პირდაპირი შეყვანა.
  - ფაილის ატვირთვა (.txt).
  - ვირტუალური კლავიატურა ეკრანზე.
- გენერირების ოპციები:
  - გენერირებული აუდიოს გაშვება.
  - გენერირებული აუდიოს `.wav` ფორმატის სახით შენახვა.
- მსუბუქი და არ იყენებს ღრუბლოვან TTS API-ს.
